### 3. S-Level Sharding — Новая концепция работает

Впервые реализовано **шардирование по S-координатам** вместо пространственного XY:
```
S-Level 0 (глобальные): ~8% нейронов  
S-Level 1 (региональные): ~15% нейронов
S-Level 2 (колонки): ~23% нейронов      ← пик распределения
S-Level 3 (микроколонки): ~31% нейронов ← доминирующий уровень  
S-Level 4 (локальные): ~24% нейронов
S-Levels 5-9: пусты (нет сверхлокальной активности)
```

Это **биологически правдоподобно** — большинство связей на уровнях 2-4 (колонки и микроколонки).

### 4. Day/Night Cycle — Чистое разделение

```
Day Phase (GPU):   физика спайков, I_SIFS расчёт, детекция спайков
Night Phase (CPU): структурная пластичность, Hebbian ΔS, resharding

Day:   0.138-0.381 ms/step  (быстро, параллельно)
Night: 0.062-0.568 ms/cycle (редко, но тщательно)
```

Архитектура **без конфликтов** — GPU не ждёт CPU, CPU работает с накопленной историей.

## Гибридные инновации (чего не было в каждом проекте отдельно)

### 1. Integer Physics + SIFS Math
```rust
type Fixed16 = i32;  // Q16.16 для детерминизма GPU
fn sifs_w(n: usize) -> Fixed16 { f32_to_fixed((-2.0 * K * n as f32).exp()) }
```

SIFS-функции переведены в fixed-point → **детерминированность на любом железе**.

### 2. S-Coordinate Sharding (революция)
```rust
neurons.s_coordinate[i] = f32_to_fixed(radius * 5.0); // φ-spiral position
let s_level = (fixed_to_f32(neurons.s_coordinate[i]) as usize).min(SIFS_LEVELS - 1);
shards[s_level].add_neuron(i);
```

Нейроны шардируются **не по физическому положению XY**, а по **масштабной координате S**. Это принципиально новая топология.

### 3. Cone Tracing + φ-Spiral
```rust
let angle = i as f32 * 2.0 * PI / (PHI * PHI);  // золотой угол
neurons.axon_direction[i] = (angle.cos(), angle.sin());
```

Аксоны растут вдоль **φ-спирали** — природная оптимальная упаковка + Cone Tracing.

### 4. SIFS Current Calculation
```rust
let contrib = (sifs_weights[n] as i64 * sigma as i64) >> 16;
i_sifs += contrib;  // 10-level summation
```

**10 порогов вместо 1** — каждый со своим весом W(n). Это даёт 3.46 bits/neuron против 1 bit.
