# Конфиг мозга (Brain TOML)

Формат конфигурации узла мозга. Соответствует [BRAIN_CONTRACT](BRAIN_CONTRACT.md) и плану п.2.4.

## Секция `[brain]`

| Параметр | Тип | По умолчанию | Описание |
|----------|-----|--------------|----------|
| `n_neurons` | uint | 1000 | Число нейронов. |
| `v0` | float | 0.02 | Базовый порог для φ-ряда V_th(n) = V₀/φⁿ. |
| `night_interval` | uint | 100 | Каждые N шагов выполняется Night Phase (Hebbian ΔS, resharding). |

## Секция `[sifs]` (опционально)

Константы K, PHI, FIB, W в коде заданы явно и синхронны с core.py. В конфиге можно зарезервировать переопределение (пока не используется).

## Пример

См. [../config/brain.example.toml](../config/brain.example.toml).

## Использование в коде

Сейчас конфиг задаётся через `BrainConfig` в Rust (значения по умолчанию или программно). Загрузка из TOML-файла будет добавлена при необходимости (зависимость `toml`).
